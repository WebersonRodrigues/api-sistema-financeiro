Api para o sistema financeiro

Esta api contém somente 2 entidades, titulos e naturezas de lançamento.

Segue a abaixo um resumo das rotas disponiveis para esta API:

Obs.: Não é necessário autenticação!
 
Referente a naturezas:
Metodos URL
GET http://localhost:3000/naturezas-lancamento
POST http://localhost:3000/naturezas-lancamento
PUT http://localhost:3000/naturezas-lancamento/{id}
   
 
Referente a titulos:
Metodos URL
GET http://localhost:3000/titulos
POST http://localhost:3000/titulos
PUT http://localhost:3000/titulos/{id}
